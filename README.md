# .github

Org-level metadata for [@karet](https://github.com/karet). Files in this
repository are surfaced across every Karet repo by GitHub as defaults.

## What lives here

| Path | What it does |
|------|--------------|
| [`profile/README.md`](profile/README.md) | Renders on the org page at <https://github.com/karet> |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Default Code of Conduct for all Karet repos |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | How to contribute, applied to repos that don't override it |
| [`SECURITY.md`](SECURITY.md) | How to responsibly disclose vulnerabilities |
| [`SUPPORT.md`](SUPPORT.md) | Where to ask questions and get help |
| [`ISSUE_TEMPLATE/`](ISSUE_TEMPLATE) | Default issue templates (bug, feature) |
| [`PULL_REQUEST_TEMPLATE.md`](PULL_REQUEST_TEMPLATE.md) | Default PR template |

A repo can override any of these by providing its own copy. See the
[GitHub docs on default community health files][gh-defaults] for the
resolution rules.

[gh-defaults]: https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file
